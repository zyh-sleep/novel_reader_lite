# 轻羽阅读器

轻羽阅读器是一个本地 TXT 小说阅读工具，主要是为了把我自己平时看网文时最常用的几个需求放到一个界面里：打开方便、切章方便、翻页顺手、下次还能接着上次的位置继续看。

我一开始只是想做一个足够轻、足够直接的阅读器，不需要联网，不需要注册，也不用装一堆额外依赖。能双击打开、导入 TXT、安心看书，就够了。

## 适合什么场景

- 本地保存了很多 `.txt` 小说，想找一个简单的阅读器直接打开
- 不想被网页广告、弹窗或者推荐流打断
- 看长篇内容时，希望能快速切章节、翻页和续读

## 目前功能

- 导入本地 `.txt` 文件
- 自动识别常见章节标题，并在左侧生成目录
- 支持章节跳转，也支持键盘左右键切换上一章 / 下一章
- 支持上下键按页翻动，在章节边界会自动衔接
- 可调字号，适合长时间阅读
- 支持日间 / 夜间模式切换
- 显示当前阅读进度
- 记录最近打开的书籍
- 自动记住上次阅读到的章节和滚动位置

## 下载后怎么用

1. 点击仓库页面右上角 `Code`
2. 选择 `Download ZIP`
3. 解压后进入项目文件夹
4. 运行下面这条命令

```powershell
python main.py
```

如果你的电脑已经装好了 Python，一般就可以直接启动。

## 快捷键

- `Ctrl + O`：打开 TXT 文件
- `Ctrl + +`：增大字号
- `Ctrl + -`：减小字号
- `Ctrl + D`：切换夜间模式
- `Ctrl + R`：显示 / 隐藏最近书单
- `↑ / ↓`：上一页 / 下一页
- `← / →`：上一章 / 下一章
