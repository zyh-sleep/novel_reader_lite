# 轻羽阅读器

轻羽阅读器是一个本地 TXT 小说阅读工具，主要是为了把我自己平时看网文时最常用的几个需求放到一个界面里：打开方便、切章方便、翻页顺手、下次还能接着上次的位置继续看。

我一开始只是想做一个足够轻、足够直接的阅读器，不需要联网，不需要注册，也不用装一堆额外依赖。能双击打开、导入 TXT、安心看书，就够了。

## 为什么会做这个项目

很多 TXT 小说其实并不缺内容，缺的是一个真正顺手的阅读方式。

我自己在看本地小说时，常见的问题一直很明显：

- 有些阅读器功能很多，但界面太重，打开一本 TXT 也要点很多步骤
- 有些工具更像文本编辑器，能打开文件，但不适合长时间沉浸式阅读
- 有些在线阅读方式干扰太多，广告、推荐、弹窗会打断阅读节奏
- 很多工具对 TXT 章节识别不稳定，切章不方便，阅读进度也不容易续上
- 有时候只是想安静看书，不想联网、不想登录，也不想折腾环境

这个项目就是针对这些问题做的。目标不是把功能堆得很复杂，而是把本地 TXT 阅读里最常用、最影响体验的几个点做好。

## 这个项目的优势

- 打开直接用：下载后解压，双击 `exe` 就能启动
- 本地优先：不依赖网络，不需要账号，不会被推荐流打断
- 阅读导向：界面重点放在章节、正文、翻页和进度，而不是编辑功能
- 操作直接：左右切章，上下翻页，适合长篇连续阅读
- 续读方便：会记住上次阅读的位置，下次打开可以直接继续
- 成本很低：体量轻，适合单纯拿来读 TXT 小说

## 适合什么场景

- 本地保存了很多 `.txt` 小说，想找一个简单的阅读器直接打开
- 不想被网页广告、弹窗或者推荐流打断
- 看长篇内容时，希望能快速切章节、翻页和续读

## 目前已经支持

- 导入本地 `.txt` 文件
- 自动识别常见章节标题，并在左侧生成目录
- 支持章节跳转，也支持键盘左右键切换上一章 / 下一章
- 支持上下键按页翻动，在章节边界会自动衔接
- 可调字号，适合长时间阅读
- 支持日间 / 夜间模式切换
- 显示当前阅读进度
- 记录最近打开的书籍
- 自动记住上次阅读到的章节和滚动位置

## 下载后怎么用

1. 点击仓库页面右上角 `Code`
2. 选择 `Download ZIP`
3. 解压后进入项目文件夹
4. 打开 `dist` 文件夹
5. 双击 `QingYuReader.exe`

整个程序已经打包好，正常情况下不需要额外安装 Python。

如果 Windows 弹出安全提示，选择“更多信息”后再点“仍要运行”即可。

## 快捷键

- `Ctrl + O`：打开 TXT 文件
- `Ctrl + +`：增大字号
- `Ctrl + -`：减小字号
- `Ctrl + D`：切换夜间模式
- `Ctrl + R`：显示 / 隐藏最近书单
- `↑ / ↓`：上一页 / 下一页
- `← / →`：上一章 / 下一章
